如何直接在github上预览html效果

实用技巧
  • github

我们经常会在Github上托管我们的web项目，但一般只能在上面看到我们的源码，如果想要在线预览我们的项目该怎么办呢？其实只需要很简单的方法就能做到了

Github项目效果在线预览

在Github上托管的项目中经常会存放一些demo的html文件在里面，比如像是下面这样的：

如果直接点击html文件只能看到html文件的源码，那么有没有什么办法可以直接在浏览器看到html的网页效果而不用克隆到本地呢？

做法是http://htmlpreview.github.com/直接把github上的html文件链接复制在上面即可，或者直接在你的html文件链接前加上http://htmlpreview.github.io/?如图：

比如，Github上我有这么一个文件https://github.com/Nicksapp/To-Do-List/blob/master/index.html,直接访问看到的是这样的：

只有这样访问我们的文件才能直接在网页中预览到效果：http://htmlpreview.github.io/?https://github.com/Nicksapp/To-Do-List/blob/master/index.html像是这样：
